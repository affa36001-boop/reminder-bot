# Reminder Bot

Персональный Telegram-помощник-напоминалка на **aiogram 3.x** с управлением через кнопки (reply + inline).

## Возможности (MVP+)

- ➕ **Создание** через FSM: календарь → часы → минуты → повтор → preview-уведомление → тег. Или одной строкой: `завтра в 15:00 встреча #работа`.
- 📋 **Список / Сегодня / Неделя** — каждое напоминание с inline-кнопками ✅ ❌ ⏰
- 🔁 **Повторения**: ежедневно, еженедельно, ежемесячно, ежегодно, каждые N дней
- 🌍 **Часовые пояса** — выбор через кнопки, время сохраняется в UTC
- 🏷 **Теги** — `#работа`, `#здоровье` и кастомные
- 🔔 **Предварительные уведомления**: за 10 мин / 1 ч / 3 ч / 1 день
- ⏰ **Отложить** на 10 мин / 1 ч / 1 день прямо из уведомления

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
# source .venv/bin/activate     # Linux/macOS
pip install -r requirements.txt
copy .env.example .env          # затем впиши свой BOT_TOKEN
python bot.py
```

## Структура

```
bot.py                 — entry point + запуск планировщика
config.py              — настройки из .env
database/
  models.py            — User, Reminder (SQLAlchemy)
  db.py                — async-сессии и CRUD-хелперы
services/
  parser.py            — разбор «завтра в 15:00», recurrence
  scheduler.py         — APScheduler tick раз в 30 сек
handlers/
  start.py             — /start, /help
  create.py            — FSM создания напоминания
  list.py              — список / сегодня / неделя
  callbacks.py         — done / del / snooze
  settings.py          — таймзона, теги
keyboards/
  reply.py             — главное меню снизу
  inline.py            — календарь, часы, минуты, повтор, теги, действия
utils/logger.py        — loguru
```

## Использование

1. `/start` → внизу появится клавиатура с кнопками
2. Тапните **➕ Создать** → введите текст (или сразу с датой)
3. Если даты нет — выберете её на календаре, потом час и минуту
4. Дальше — повтор, pre-notify, тег
5. В назначенное время бот пришлёт сообщение с кнопками **✅ Выполнено / ❌ Удалить / ⏰ Отложить**

## Команды (fallback к кнопкам)

`/start`, `/help`, `/new`, `/list`, `/today`, `/week`, `/settings`, `/timezone`, `/cancel`
